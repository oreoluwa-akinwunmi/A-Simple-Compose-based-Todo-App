# A-Simple-Compose-based-Todo-App

# Project Overview
A Todo List Application that allows:
- Adding a task
- Deleting a task
- Marking a task as completed

# How Compose Works
Compose uses composables, similar to Flutter widgets to design the UI of an application in a tree format. They use composables (functions that describe the UI) and states to keep track of changes.
